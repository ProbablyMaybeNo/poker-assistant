Poker Assistant Icons
=====================

Place your icon files here:

- poker_icon.ico  - Windows icon file (256x256, 128x128, 64x64, 48x48, 32x32, 16x16)
- poker_icon.png  - PNG fallback (256x256)

Icon Requirements:
- ICO format for best Windows compatibility
- Multiple sizes embedded for different contexts
- Poker/card theme recommended (e.g., card suits, chips, etc.)

Creating an Icon:
1. Design or find a poker-themed icon (256x256 PNG)
2. Use an online converter like:
   - https://icoconvert.com/
   - https://convertio.co/png-ico/
3. Save as poker_icon.ico in this directory

The application will work without an icon but will use system defaults.
